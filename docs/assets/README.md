## 🎨 Design System & Branding Guidelines

When generating or replacing screenshots in this directory, adhere to the Sentinel OS aesthetic standards:
- **Theme:** Dark Mode Glassmorphism (Background: `#0B0F19`, Surface: `#111827` with backdrop blur).
- **Accent Colors:**
  - ⚡ **Electric Cyan / Blue (`#3B82F6` / `#00F0FF`):** System telemetry and active LangGraph transitions.
  - 🚨 **Alert Red (`#EF4444`):** Anomaly Z-Score > 3.0 and stockout warnings.
  - ✅ **Emerald Green (`#10B981`):** Operator approved actions and healthy container status.
- **Typography:** Inter / Outfit sans-serif with clean tabular numbers for supply chain metrics.
